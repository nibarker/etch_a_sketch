Etch-A-Sketch Project

"...a browser version of something between a sketchpad and an Etch-A-Sketch."

Each mouse pass in the Pencil Grid darkens the grid squares by 10%. Each mouse pass in the Colorful Grid colors the grid with a random color.

Reset the grid using the buttons. The size entered at the prompt gives the number of squares in the column and the row. Use the mouse to draw on the grid.
